# To-Do List Application

This is a simple To-Do List application that allows you to add, complete, and delete tasks. It's built with Flutter, a popular open-source framework for building natively compiled applications for mobile, web, and desktop from a single codebase.

## Features

- Add new tasks to your to-do list.
- Mark tasks as completed when you're done.
- Easily delete tasks you no longer need.
- Search functionality to filter tasks.
- Beautiful and user-friendly interface.

## Installation

To run this application on your local machine, follow these steps:

1. Clone this repository to your local machine.
2. Make sure you have Flutter and Dart installed. If not, you can [get them here](https://flutter.dev/docs/get-started/install).
3. Open the project in your preferred Flutter development environment.
4. Run the application on your desired platform (iOS, Android, web, etc.).

## Usage

1. Launch the app.
2. Click on the "+" button to add a new task.
3. Tap on a task to mark it as completed.
4. To delete a task, swipe it to the left or right.
5. Use the search bar to filter tasks by keyword.
